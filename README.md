# Agente de IA - Mentor de Carreira em Tecnologia

## Sobre o Projeto

Projeto desenvolvido durante a jornada de estudos da DIO com o objetivo de criar um agente de Inteligência Artificial capaz de orientar estudantes e profissionais que desejam iniciar ou migrar suas carreiras para a área de tecnologia.

## Objetivos

- Compreender a estrutura de agentes utilizando CrewAI;
- Aplicar conceitos de IA Generativa;
- Utilizar ferramentas de apoio ao desenvolvimento assistido por IA;
- Desenvolver um mentor virtual para orientação profissional.

## Tecnologias Utilizadas

- Python
- CrewAI
- Ollama
- OpenCode
- GitHub

## Estrutura do Agente

### Role
Mentor de Carreira em Tecnologia

### Goal
Auxiliar alunos a identificarem carreiras alinhadas ao seu perfil, experiência e objetivos profissionais.

### Backstory
Mentor experiente em tecnologia com conhecimento sobre mercado de trabalho e trilhas de aprendizagem da DIO.

## Funcionamento

O agente recebe informações sobre o perfil do usuário e consulta as carreiras disponíveis na DIO para gerar recomendações personalizadas.

## Aprendizados

Durante este projeto foi possível compreender:

- Conceito de Agentes de IA;
- Estrutura Role, Goal e Backstory;
- Engenharia de Prompts;
- Integração entre IA e Python;
- Utilização de ferramentas de automação para desenvolvimento.
